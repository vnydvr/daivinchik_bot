# daivinchik_bot

написанный мною автофильтр, по-моему мнению, мусорных анкет в дайвинчике
ФУНКЦИИ:
- банворды(автодизлайк)
- гринворды(звуковой сигнал и показ меню действий)
- блеклист(анкеты оттуда скипаются автоматически)
- желаемый возрастной диапазон
- желаемая длина текста анкеты
- фильтр анкета-системное сообщение(при появлении второго вызывается пользователь)
- скрость примерно равна 5-10 анкетам в секунду
- автодизлайк анкетам без описания
